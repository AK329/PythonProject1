(For pycharm)
প্রজেক্টটি চালানোর সম্পূর্ণ প্রক্রিয়া (Step-by-Step Guide)

বিচারকদের মূল্যায়নের সুবিধার্থে এই প্রজেক্টটি আপনার লোকাল পিসিতে চালানোর পূর্ণাঙ্গ ধাপসমূহ নিচে দেওয়া হলো:

### ১. পূর্বশর্ত (Prerequisites)

নিশ্চিত করুন যে আপনার পিসিতে **Python (3.9+)** এবং **Git** ইনস্টল করা আছে।

### ২. রিপোজিটরি ক্লোন বা ডাউনলোড করুন (Clone the Project)

প্রথমে টার্মিনাল বা কমান্ড প্রম্পট ওপেন করে নিচের কমান্ডটি লিখুন:

```bash
git clone https://github.com/AK329/MediGuardian-AI.git
cd MediGuardian-AI

```

### ৩. প্রয়োজনীয় লাইব্রেরি ইনস্টল করুন (Install Dependencies)

সিস্টেমটি চালানোর জন্য প্রয়োজনীয় সব লাইব্রেরি (Technical Reliability নিশ্চিত করতে) ইনস্টল করুন:

```bash
pip install -r requirements.txt

```

*দ্রষ্টব্য: এটি `streamlit`, `pandas`, `scikit-learn` এবং `numpy` ইনস্টল করবে।*

### ৪. এআই মডেল ট্রেইন করুন (Train the AI Model)

সিস্টেমের "ব্রেইন" বা AI মডেলটি তৈরি করতে নিচের কমান্ডটি রান করুন:

```bash
python ML_Trainer.py

```

*এর ফলে `medi_guardian_model.pkl` নামে একটি ফাইল তৈরি হবে যা আপনার অ্যাপের সিদ্ধান্ত গ্রহণে সাহায্য করবে。*

### ৫. অ্যাপ্লিকেশনটি রান করুন (Launch the App)

সবশেষে আপনার ড্যাশবোর্ডটি ওপেন করতে নিচের কমান্ডটি দিন:

```bash
streamlit run app.py

```

*কমান্ডটি দেওয়ার পর আপনার ব্রাউজারে একটি লোকাল লিঙ্ক (যেমন: `localhost:8501`) ওপেন হবে।*
